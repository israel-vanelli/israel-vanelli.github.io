2. Tables

Table Tags

Tag Name Description
<table> Table The wrapper element for all HTML tables.
<thead> Table Head The set of rows defining the column headers in a table.
<tbody> Table Body The set of rows containing actual table data.
<tr> Table Row The table row container.
<td> Table Data The table row container.
<tfoot> Table Foot The set of rows defining the footer in a table.

Table Attributes

Attribute Name Description
colspan Column Span Defines how many columns a td element should span.
rowspan Row Span Defines how many rows a td element should span.

3. Forms

Forms are utilized to submit information, using controls which have a {"name":"value"} pair format when sending or requests with the provided data.

The <form> Element

Forms have two main attributes: id and method. The id attribute contains the form unique identifier, which can be used to link <input> elements (further explanation below), and the method attribute is used to define the HTTP Method this form will be using. TO DO: HTTP Cheatsheet.

<input> Element Types

Input elements are used to accept user submitted data, and can be linked to a form by being nested inside the form, or being linked with the <form> attribute.

Below are listed a few common types of <Input> elements.

Type Name Appearance
"text" Text Field
"range" Range
"checkbox" Check Box
"radio" Radio
Check MDN web docs for more info on <input>.

<input> Attributes

Attribute Description
form The attribute in which is specified the associated form id of the <input> element. If none is specified then the <input> element is associated with the nearest available form.
id The Unique Identifier attribute, can be referenced in the for attribute in <label>.
name The "name" of the control in the {"name":"value"} pair, can be used to group "checkbox" or "radio" controls.
value The "value" of the control in the {"name":"value"} pair, sets the initial value of the element.
max and min Can be used to set the maximum and minimum values accepted by the control.
maxlength and minlength Can be used to set the maximum length and minimum length (number of characters) of the control.
required Used to specify that the field is obligatory.
