The flex-grow attribute sets how much the defined flex items will grow relative to the other flex items inside the same container.

In the example above the Green item has a flex-grow attribute with the value of 3, while Red and Blue have a value of 1, resulting in the Green item growing 3 times more than the Red and Blue items.

The flex-shrink attribute sets how much the defined flex items will shrink relative to the other flex items inside the same container.

In the example above the Green item has a flex-shrink attribute with the value of 3, while Red and Blue have a value of 1, resulting in the Green item shrinking 3 times more than the Red and Blue items. All flex items have a default flex-shrink value of 1.

Specifies the initial length of a flex item. In this example the flex-basis of all the items is 350px.


flex:

Shorthand property for flex-grow flex-shrink flex-basis;

Examples:

flex: 1 2 150px;
equivalent to flex-grow: 1; flex-shrink: 2; flex-basis: 150px;


flex: 1 2;
equivalent to flex-grow: 1; flex-shrink: 2; But no flex-basis is specified.


flex: 2 150px;
equivalent to flex-grow: 2; flex-basis: 150px;


Note: there is no way to set only flex-shrink and flex-basis with only two values.


flex-wrap:

The property used to define if flex items are forced to stay in the current row or should move to the next line if necessary. The default value for this property is nowrap.


flex-wrap: wrap

Flex items will move to another row if the container isn't wide enough to fit them.

Flex items will move to another row if the container isn't wide enough to fit them, but the direction and the order of which the items are placed will be inversed.

Since the direction is reversed, flex items will move to the previous row.
